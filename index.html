OpenMockup A standard interchange file format and domain model for mockups

I'm creating an standard for interchanging mockup files across systems, and a Domain Model to give a language.

I'm seeking to first create a community, then do this together.

Join the community by following @OpenMockup at twitter

The community is officially made public today so, if you find you are the first follower it could be pretty normal ;)
